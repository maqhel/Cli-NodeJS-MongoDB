## Tasks CLI

Tasks CLI is a program to manage your tasks in a database using terminal or console. This is a sample project for beginners

### Requirements

- Nodejs
- Mongodb Local Installation or Cloud

### Installation

```sh
npm install
```

```sh
npm link
```

### Commands

```
taskcli --version
taskcli -V
```

```
taskcli --help
taskcli -h
```

```
taskcli list
taskcli l
```

```
taskcli find <title>
taskcli f <title>
```

```sh
taskcli save
taskcli s
```

```sh
taskcli update <id>
```

```sh
taskcli delete <id>
taskcli d <id>
```
